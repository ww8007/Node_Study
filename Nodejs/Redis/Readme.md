# Redis

### Redis란

메모리에 데이터를 저장
C언어로 작성

DB처럼 사용가능

- 용도
  1. 사용자의 세션 서버
  2. 웹페이지 캐싱
     - 사이트 접속 시 미리 데이터를 적재 해둬서 속도 빠르게 하기

* 서버를 여러개 둬서 하나의 redis 서버를 바라보게 함

### 설치

brew install redis

docker run -p 6379:6379 redis

### 사용처

- url은 하나지만 서버는 달라지는 경우가 있음 - 채팅 서버
  세션 서버를 여기에 사용, 인증 관련

* docker 설치하기 싫은 경우

  1. redis-server
  1. redis-cli

* docker 진행 하는 이유
  윈도우에서 하위 버전 설치 지원 안하는 것과 공식 버전을 설치 할 수 있기 때문에 좋음
